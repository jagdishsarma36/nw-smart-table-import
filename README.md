Import Excel, CSV, TSV, and HTML tables directly into the Classic Editor.

== Features ==

- Excel paste support
- CSV support
- TSV support
- HTML table support
- Multi-table support
- TinyMCE integration
- Insert at cursor position
- Preserve formatting

== Installation ==

1. Upload plugin folder
2. Activate plugin
3. Open Classic Editor
4. Use "Import Table" button
